# Установка и настройка

Ставится как любой плагин для Revit — пять минут, и на ленте появляется
вкладка **ERP**.

## Что нужно

- Windows 10 или 11
- Autodesk Revit 2023 или 2024
- .NET Framework 4.8+ (почти наверняка уже стоит вместе с Windows или Revit)

## Скачайте

Откройте [Releases](../../../releases) и возьмите файл под свою версию Revit:

- `ElectricityRevitPlugin_R23_..._x.x.x` — для Revit 2023
- `ElectricityRevitPlugin_R24_..._x.x.x` — для Revit 2024

Есть два формата: **MSI** (обычный установщик) и **ZIP** (если нет прав
администратора или хочется разложить файлы самостоятельно).

## Установите

**MSI** — запустите файл от имени администратора и пройдите мастер установки.

**ZIP** — распакуйте архив и запустите `Install.bat` от имени администратора:
он сам разложит всё по нужным папкам.

### Нет прав администратора?

Файлы можно разложить руками:

1. Содержимое папки `Plugin` → `%APPDATA%\Autodesk\Revit\Addins\<год>\ElectricityRevitPlugin\<версия>\`
2. `Addin\ElectricityRevitPlugin.addin` → `%APPDATA%\Autodesk\Revit\Addins\<год>\`
3. Откройте скопированный `.addin` в текстовом редакторе и проверьте, что путь
   в теге `<Assembly>` указывает на
   `ElectricityRevitPlugin\<версия>\ElectricityRevitPlugin.Application.dll`
4. Запустите Revit

(`<год>` — версия Revit, `<версия>` — номер версии плагина из имени файла)

## Готово

Открывайте Revit — на ленте вас ждёт вкладка **ERP** со всеми инструментами.
→ [Что там есть](ribbon-commands.md)

## Обновления — сами по себе

При каждом старте Revit плагин тихо проверяет, не вышла ли новая версия, и
подтягивает её в фоне, не мешая работе. Новая версия включится при следующем
перезапуске Revit — нажимать ничего не нужно.

Если хотите отключить (например, на строго контролируемых рабочих местах) —
переменная среды:

```
ERP_DISABLE_AUTOUPDATE=1
```

## Передумали?

1. Удалите папку `%APPDATA%\Autodesk\Revit\Addins\<год>\ElectricityRevitPlugin\`
2. Удалите файл `ElectricityRevitPlugin.addin` из
   `%APPDATA%\Autodesk\Revit\Addins\<год>\`
