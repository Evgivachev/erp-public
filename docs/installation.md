# Установка и настройка

## Системные требования

- Windows 10 или 11.
- Autodesk Revit 2023 или 2024.
- .NET Framework 4.8 или новее (обычно уже установлен вместе с Windows/Revit).

## Скачивание

Зайдите в раздел [Releases](../../../releases) этого репозитория и скачайте файл
под свою версию Revit:

- `ElectricityRevitPlugin_R23_..._x.x.x` — для Revit 2023
- `ElectricityRevitPlugin_R24_..._x.x.x` — для Revit 2024

Доступны два формата: **MSI** (обычный установщик) и **ZIP** (если нет прав
администратора на установку MSI или нужен ручной контроль над файлами).

## Автоматическая установка

**MSI**: запустите файл `.msi` от имени администратора и пройдите мастер
установки.

**ZIP**: распакуйте архив и запустите `Install.bat` от имени администратора —
он сам разложит файлы по нужным папкам.

## Ручная установка (из ZIP)

Если нет прав администратора — можно скопировать файлы вручную:

1. Скопируйте содержимое папки `Plugin` в:
   `%APPDATA%\Autodesk\Revit\Addins\<год>\ElectricityRevitPlugin\<версия>\`
2. Скопируйте файл `Addin\ElectricityRevitPlugin.addin` в:
   `%APPDATA%\Autodesk\Revit\Addins\<год>\`
3. Откройте скопированный `.addin`-файл текстовым редактором и убедитесь,
   что путь в теге `<Assembly>` указывает на
   `ElectricityRevitPlugin\<версия>\ElectricityRevitPlugin.Application.dll`.
4. Запустите Revit.

`<год>` — год версии Revit (2023 или 2024), `<версия>` — номер версии плагина
из имени скачанного файла.

## Первый запуск

После установки в Revit появится новая вкладка **ERP** с инструментами
плагина — см. [обзор команд на ленте](ribbon-commands.md).

## Автообновление

Плагин сам следит за новыми версиями: при каждом старте Revit он тихо
проверяет GitHub Releases и, если вышла новая версия, скачивает её в соседнюю
подпапку — работа с текущей версией при этом не прерывается. Новая версия
включается при следующем перезапуске Revit. Никаких действий от вас не
требуется.

Если нужно отключить автообновление (например, на строго контролируемых
машинах), задайте переменную среды:

```
ERP_DISABLE_AUTOUPDATE=1
```

## Удаление

1. Удалите папку плагина со всеми версиями:
   `%APPDATA%\Autodesk\Revit\Addins\<год>\ElectricityRevitPlugin\`
2. Удалите файл `ElectricityRevitPlugin.addin` из
   `%APPDATA%\Autodesk\Revit\Addins\<год>\`
