# Обычная работа: команды на ленте

Никакого ИИ, только кнопки. После установки в Revit появляется вкладка
**ERP** — инструменты сгруппированы по смыслу, как на любой привычной панели.
Ниже — что делает каждая группа.

## Цепи

- **Маркировка электрических систем** — автоматическая нумерация и обновление
  марок цепей по выделенным элементам.
- **Кабельный журнал** — формирование кабельного журнала по модели.
- **Распределение по фазам** — равномерное распределение отходящих линий по
  фазам.
- **Расчёт токов КЗ** — расчёт токов короткого замыкания для проверки защитных
  характеристик.
- **Подсчёт светильников в помещениях** — количество светильников по каждому
  помещению.
- **Обновление параметров** — массовое обновление параметров цепей и щитов
  (например, после правок в модели).
- **Высота установки** — управление отметками установки элементов относительно
  уровня.

## Схемы

- **Однолинейная схема** — создание и обновление однолинейных схем щитов.
- **Общая предметная диаграмма (ВРУ)** — построение схемы вводно-распределительного
  устройства.
- **Разрезы кабельных лотков** — автоматическое построение разрезов лотков с
  кабелями.

## Щиты

- **Подбор корпуса щита** — подбор модели корпуса щитового оборудования по
  составу цепей.

## Нагрузки

- **Экспорт нагрузок в Excel** — выгрузка электрических нагрузок в таблицу.
- **Импорт нагрузок из Excel** — обратная загрузка расчётов нагрузок в модель.

## Общая

- **Печать** — пакетная печать листов с настройками.
- **Экспорт спецификаций** — автоматическая выгрузка ведомостей и спецификаций.
- **Добавление параметров в семейства** — массовое добавление параметров в
  выбранные семейства.
- **Редактирование координат элементов** — точная правка положения и поворота
  элементов.

## Листы

- **Сортировка листов** — упорядочивание листов в проекте.
- **Дублирование листов** — создание копий листов с видами.

## Освещение

- **Экспорт в STF / импорт из Dialux** — обмен данными о светильниках со
  светотехническими программами (Dialux и др.).
- **Выравнивание светильников по центру помещения / по прямоугольной сетке** —
  автоматическая раскладка светильников в границах помещения.
- **Маркировка аварийного освещения** — расстановка марок для аварийных
  светильников.
