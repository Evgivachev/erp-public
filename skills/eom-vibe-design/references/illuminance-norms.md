# Нормируемая освещённость по назначению помещений

Ориентировочные значения средней горизонтальной освещённости (лк) по
СП 52.13330.2016 «Естественное и искусственное освещение» для типовых
помещений. Плоскость нормирования — рабочая поверхность Г-0.8 (0,8 м от пола),
если не указано иное.

**Это шпаргалка для первого приближения.** Итоговое значение всегда
подтверждай у пользователя: разряд зрительной работы, задание на
проектирование или ведомственные нормы могут давать другое значение.

| Помещение | Освещённость, лк |
|---|---|
| Кабинеты, офисы, рабочие комнаты | 300 |
| Проектные залы, конструкторские бюро | 500 |
| Конференц-залы, залы заседаний | 200 |
| Читальные залы | 300–400 |
| Учебные классы, аудитории | 400 |
| Вестибюли, холлы | 150 |
| Коридоры, проходы (общественные здания) | 75–100 |
| Лестничные клетки | 50–100 |
| Санузлы, умывальные | 75 |
| Гардеробные | 150 |
| Кладовые, подсобные | 50–75 |
| Серверные, аппаратные | 400 |
| Электрощитовые, венткамеры, тех. помещения | 20–50 (дежурное — по заданию) |
| Торговые залы | 300–500 |
| Жилые комнаты, кухни | 150 |
| Ванные в жилье | 50 |
| Закрытые автостоянки | 75 |

Если назначение помещения из его имени/номера не очевидно — спроси
пользователя, не подбирай норму по догадке.

## Оценка количества светильников (первое приближение)

N = (E × S × Kз) / (Φсв × U), где E — норма (лк), S — площадь (м²),
Kз — коэффициент запаса (1.1–1.4), Φсв — световой поток светильника (лм),
U — коэффициент использования (0.4–0.6 для типовых офисных помещений).

Полученное N округляй вверх до числа, ложащегося в равномерную сетку
(`CalculateFixtureLayout` сделает раскладку по контуру помещения); после
расстановки сообщи пользователю фактическую расчётную освещённость.
